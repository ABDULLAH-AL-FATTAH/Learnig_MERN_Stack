Git Repository create & initialization

  1. go to github and create Repository
  2. make a folder as same of your repo name
  3. go to vscode and click ctrl+j to open terminal from vscode
  4. git init
  5. git add .
  6. git commit -m"first commit"
  7. git branch -M main
  8. git remote add origin https://github.com/fattahfolder/my-first-repo.git
  9. git push -u origin main
  10. N.B: when you create a new Repository in the Github, you've to do the above the steps

Git Incremental Code

  • git add .
  • git commit -m"add a meaningful commit message"
  • git push

To make your live website through github

  • Go to Settings
  • Go to Pages
  • Select Branch> Main
  • Now go to the git ripo and open html file
  • Then go to the URL box
  • https://username.github.io/path link

Common github related issues for beginners

  1. Wrong Repository: You'll get errors
  2. Try to "git commit" before "git add ."
  3. You can't solve any error, if you don't read the errer message carefully

[Advanced] Multiple developer workflow contributor on Private Repository

  1. Create Repository
  2. Select Private then click Create
  3. As usually create a folder name like the Repository Name
  4. Go to the folder and go to the folder path and cmd enter
  5. As usually Git Repository create & initialization from VSCode (ctrl+j)
  6. if you want to permit other person to work on your code
  7. Then go to Github Settings> collaborator> confirm your github password
  8. add collaborator username or his E-mail
  9. git clone: go to 'Code' box on Github>HTTPS>copy the path
  10. Then paste the link to your cmd to the specific folder

[Advanced] git branch, pull request, merge

  • If want to contribute your idia to other git project
  • Then click git fork and check "copy the main brach only"> click 'create fork'
  • Now you got the project by forked from the other user on your git Repositories List
  • Now clone it to your PC
  • Now open the project by cmd (code .) to update the project code
  • Now check your git branch from vscode terminal
  • If it show 'main' then you've to create new branch
  • 'git checkout -b add-branch_name' from vs terminal
  • Now check your git branch again
  • if it shows '*add-branch_name' now you can write your code
  • After write your code then follow the incremental code of git(git add ., git commit -m"update", git push)
  • when you try to 'git push' then it shows fatal, read the message carefully
  • Now the comman is 'git push --set-uptream origin add-fruits' that's it
  • So, now go to Github Repository then you will see "compare & pull request" to send you code to the main branch
